# rd-160052-JS-Port


JS Port of minecraft version rd-160052 with three.js as renderer

![Preview](https://raw.githubusercontent.com/Fraeric123/rd-160052-JS-Port/main/screens/s1.png)

## Controls
* WASD        - Movement
* Space       - Jump
* R           - Random Teleport
* G           - Spawn Steve
* O           - Take screenshot
* Enter       - Save level.dat
* INSERT      - Load level.dat
* Mouse wheel - Select block
* LMB         - Destroy block
* RMB         - Place block
* 1           - Rock
* 2           - Dirt
* 3           - Stone Brick
* 4           - Wood
